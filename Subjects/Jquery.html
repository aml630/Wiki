Jquery

Jquery is the most popular Javascript library. It helps you accomplish javascript tasks more quickly and easily. Most projects you use involving javascript will utilize jquery.

The following is a list of common problems you'll use jquery to help you with

  • HTML DOM traversal
  • DOM Manipulation
  • Event handling
  • Ajax

Like all libraries jquery must be linked to at the beginning of your HTML. Your personal scripts file must come after this library link.
