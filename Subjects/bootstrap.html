Bootstrap

Bootstrap is one of the most effective and popular frameworks for simplifying your initial stylization process and ensuring your page is functional on many different screen sizes

Bootstrap can be used for the vast majority of a web develops needs when building out their HTML

Below is a short list of the most popular tools bootstrap provides

  • An easy grid system for HTML element placement
  • Easy professional looking stylization of all HTML elements
  • Easy animation and functionality of basic HTML functions

Grid System

The grid system is one of the many aspects within Bootstrap. It allows you to neatly divide your context into rows and columns.

The grid system allows you to formant and divide your page into many different ways. You can have 12 columns, 4 columns, 3 columns etc.

Columns become responsive and adjusts to the size of the screen.
