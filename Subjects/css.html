CSS

CSS is a language used to style pages

  • Change font types and sizes
  • Add padding, margins, and borders
  • Center text and images
  • Create floats
  • Target specific items with cascading

Cascading Styles

Cascading styles are able to target elements, classes and ID's within eachother in any order.

This allows the designer to be very specific about where the design being created will show

This also allows a greater degree of organization within both the HTML and the stylesheets.

Block Elements

Block Elements are tags within html. They are used to create headers with the <'h'> tag, paragraphs with the <'p'> tag, and lists with <'ul'> and <'ol'> tags.

Some other tags are:

  • img for images
  • divs

Inline Elements

Inline elements are like block elements in that they help you form your HTML. However, they don't start on a new line and instead of creating space for themselves they try and take up a little room as possible to accomplish their objective.

In other words, inline elements help you shape the inside of block elements

Some examples of inline elements include

  • Span Tags
  • Img Tags/li>
  • A tags
  • Strong tags
  • Em tags

Divs and Spans

A div is basically a divsion of space or section in HTML space. They are used to group block elements together.

A span will target a specific section within a div.

Both divs and spans are given specific class

Box Model

The box model allows the user to create space around their content. It allows user to set specific pixel sizes to the margin, padding, and border.

Floats

In CSS the float property is a way to position different elements within the page.

By alloiwng an element to float you can easily position it on either side of the page with other elements in your text wraping around it
